## 口袋晋城 食用指南

#### 页面结构

分为两类页面, 目录页和内容页.

均为根据 https://www.teambition.com/research/join 页面改造.

#### 文件目录结构

```
-- index.html
-- page-1.html
-- page-2.html
-- page-3.html
-- page-4.html
-- page-5.html
-- main.css
-- images/
-- images/content/
```

页面组件图片放入 `images/` 内, 展现内容的图片放入 `images/content/` 内.
